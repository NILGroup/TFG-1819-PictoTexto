\section{Trabajo Individual de Jose Maria}
%-------------------------------------------------------------------
\label{cap6:sec:Jose Maria}

Lo primero que realice fue un estudio de los SAACS, poniendo especial atención en los pictogramas, y los Servicios Web. 

En la búsqueda de información sobre los SAACS, analice los sistemas pictograficos más extendidos y la comunicación a través de  pictogramas. 

Uno de los requisitos indispensables de este proyecto, era la necesidad de incluir la aplicación dentro de un Servicio Web, por lo cual investigue sobre cuales eran las diferentes opciones para crear un Servicio Web en Python. Además desarrolle una prototipo tecnológico, para aprender a crear servicios web con Python.

A continuación, pensamos en la metodologia de trabajo que seguiriamos y la documente  en la memoria en el capitulo \ref{cap4:MetodologiaDeTrabajo}.

El siguiente paso necesario, era un estudio sobre los diferentes sistemas de generación de lenguaje natural parte fundamental del proyecto. Investigue y documente la base teórica de la generación de lenguaje natural, una vez termine esto pase a la búsqueda de diferentes frameworks de generación de lenguaje natural en español, tras la cual encontré el framework SimpleNLG-ES. Una vez presentados los resultados de mi investigación en la memoria, desarrolle un prototipo tecnológico utilizando dicho framework. Este prototipo muy simple permitía enviarle un sujeto, un verbo y un complemento directo y este devolvía una frase en lenguaje natural con dichos elementos.
Para generar este servicio web, analice diferentes vías de servicios web Java, y realice dos prototipos uno con WildFly\footnote{http://www.wildfly.org/} y otro con Apache Tomcat\footnote{http://tomcat.apache.org/}, y teniendo en cuenta diferente factores como rendimiento o la comunidad decidí implementar Tomcat. Durante el desarrollo de este prototipo surgió el problema de recibir datos en formatos JSON en un servicio web Java, tras analizar las diferentes posibilidades que encontré decidí optar por usar el framework GSON.

Una vez termine el trabajo de implementación, documente los servicios web y los principios de la arquitectura, en la memoria.

Dado el limite del framework Django para la gestión de aplicaciones web desde el lado del cliente, desarrolle una aplicación web fron-end en Angular CLI, que después integre con la aplicación Django, para de esta manera obtener mayor funcionalidad, flexibilidad y descargar parte del trabajo en el cliente. Una vez terminada la implementación documente la aplicación, en la memoria. 

Después realice un estudio sobre los diferentes analizadores de lenguaje natural para el castellano y decidimos usar Spacy. A continuación de investigar su uso, desarrolle el servicio web de detección de palabras y posteriormente documente su funcionamiento.  Además de este servicio decidimos que era necesario, crear un servicio web que detectara el tiempo verbal en el que se debería conjugar la frase. Con la respuesta de este servicio se llamaría a un servicio web u otro dentro del servicio de generación de lenguaje natural. Una vez estudiados los pasos a seguir desarrolle los tres servicios web. Para los servicios de generación de lenguaje tuve que hacer una refactorización el servicio ya creado, para abstraer el comportamiento común de los diferentes servicios web de la generación de lenguaje. Una  vez terminado todo los descrito documente los servicios web.
